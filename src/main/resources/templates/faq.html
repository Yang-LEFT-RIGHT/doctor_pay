账号与登录问题

5个问题

Q1: 忘记密码怎么办？

如果您忘记了密码，请按照以下步骤操作：

  1. 点击登录页面的"忘记密码"链接
  2. 输入您的注册邮箱地址
  3. 查收邮件中的密码重置链接
  4. 点击链接设置新密码
如果未收到重置邮件，请检查垃圾邮件文件夹或联系技术支持

Q2: 如何修改个人资料？

修改个人资料的步骤：

  1. 登录系统后，点击顶部导航栏的"个人信息"
  2. 在个人信息页面，点击"编辑"按钮
  3. 修改需要更新的信息（如联系方式、专业方向等）
  4. 点击"保存更改"完成更新

Q3: 账号被锁定怎么办？

如果连续5次输入错误密码，账号将被临时锁定30分钟。

解决方法：

  1. 等待30分钟后重试登录
  2. 如急需登录，请联系管理员解锁账号
  3. 通过"忘记密码"功能重置密码

任务管理问题

7个问题

Q4: 如何接受新的任务？

接受任务的完整流程：

  1. 登录系统进入个人首页
  2. 查看"待处理任务"区域
  3. 点击任务名称查看详情
  4. 仔细阅读任务要求和截止日期
  5. 确认无误后点击"接受任务"按钮
注意：接受任务后不可随意取消，请谨慎操作

Q5: 任务进度如何更新？

更新任务进度的两种方式：

方式一：通过任务详情页

  1. 进入"任务列表"页面
  2. 点击需要更新的任务
  3. 在任务详情页找到"更新进度"按钮
  4. 填写当前完成情况和遇到的问题
  5. 提交更新

方式二：通过工时记录

  1. 每次记录工时后，系统会自动更新任务进度
  2. 完成的工作时长会累计到任务总进度
  3. 当达到预计工时，任务状态自动更新

Q6: 如何申请任务延期？

如遇特殊情况需要延期，请按以下步骤操作：

  1. 在任务到期前至少3天提出申请
  2. 进入任务详情页面
  3. 点击"申请延期"按钮
  4. 填写延期原因和新的预计完成时间
  5. 提交申请，等待管理员审核
延期申请需要管理员批准，请提供充分的理由

工时记录问题

4个问题

Q7: 如何记录工作时间？

记录工作时间的步骤：

  1. 进入"任务列表"页面
  2. 找到正在执行的任务
  3. 点击"记录工时"按钮
  4. 填写工作日期、开始时间、结束时间和工作内容
  5. 提交记录
建议每次完成工作后立即记录，避免遗漏

Q8: 工时记录可以修改吗？

工时记录在特定条件下可以修改：

可以修改的情况：

  • 记录时间错误（在24小时内）
  • 忘记记录需要补记（在3天内）
  • 工作内容描述不准确需要修改

不可修改的情况：

  • 记录已超过24小时
  • 记录已被管理员审核通过
  • 任务已结束并结算
如需修改已超过期限的记录，请联系管理员申请修改

报告系统问题

3个问题

Q9: 如何生成月度工作报告？

生成月度报告的步骤：

  1. 进入"个人统计"页面
  2. 选择需要生成报告的月份
  3. 点击"生成月度报告"按钮
  4. 系统会自动生成包含以下内容的报告：
    • 本月完成的任务清单
    • 工时统计和图表
    • 工作内容总结
    • 绩效评估
  5. 预览报告内容，确认无误后下载

Q10: 报告格式有哪些？

系统支持多种报告格式：

PDF格式

适合打印和正式提交，格式固定

Excel格式

包含详细数据，适合进一步分析

Word格式

可编辑，适合添加额外说明

建议使用PDF格式进行正式提交，Excel格式用于数据分析

技术问题

6个问题

Q11: 系统无法登录怎么办？

如果遇到登录问题，请按以下顺序排查：

  1. 检查网络连接是否正常
  2. 确认网址是否正确（https://docim.your-university.edu）
  3. 清除浏览器缓存和Cookie
  4. 尝试使用其他浏览器（推荐Chrome或Firefox）
  5. 检查账号是否被锁定（连续输错密码）

如仍无法解决，请联系技术支持：

  • 邮箱：support@docim.edu
  • 电话：010-12345678
  • 工作时间：工作日 9:00-17:00

Q12: 页面加载缓慢怎么办？

优化页面加载速度的方法：

用户端优化：

  • 清除浏览器缓存和临时文件
  • 关闭其他占用带宽的程序
  • 使用最新版本的浏览器
  • 检查网络连接质量

如果问题持续存在：

  • 可能是服务器维护或高峰期
  • 请稍后重试或联系技术支持
  • 报告问题时提供以下信息：
    • 浏览器类型和版本
    • 具体的错误信息
    • 出现问题的页面和操作
