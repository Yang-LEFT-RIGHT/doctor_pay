博士生管理系统用户手册

全面详细的系统使用指南，帮助您快速掌握系统操作流程

系统简介

欢迎使用博士生管理系统！本系统旨在为博士生和管理者提供一个高效、便捷的任务管理和津贴发放平台。

核心功能

  • 任务管理：发布、确认、跟踪和管理各类任务
  • 津贴管理：自动计算和发放博士生津贴
  • 统计分析：提供详细的数据报表和可视化图表
  • 个人中心：统一管理个人信息和安全设置

系统角色说明

角色 主要功能 权限说明
博士生 任务确认、个人资料管理、报表查看 查看和确认分配给自己的任务，管理个人信息
管理员 任务发布、用户管理、统计分析 发布和管理任务，查看全局统计，管理用户信息

系统要求

浏览器兼容性

为确保最佳使用体验，建议使用以下浏览器的最新版本：

浏览器 推荐版本 支持状态
Google Chrome 90+ 完全支持
Mozilla Firefox 88+ 完全支持
Safari 14+ 完全支持
Microsoft Edge 90+ 完全支持

重要提示

Internet Explorer 浏览器已不再支持，请使用其他现代浏览器访问系统。

网络要求

  • 稳定的互联网连接
  • 建议使用校园网或高速网络环境
  • 系统支持移动设备访问，但部分功能在桌面端体验更佳

首次登录

登录流程

访问系统

通过系统提供的网址访问登录页面，输入您的账号和初始密码。

修改密码

首次登录后，系统会提示您修改初始密码。新密码需包含字母、数字和特殊字符，长度至少8位。

完善个人信息

登录后请立即完善个人资料，特别是银行账户信息，以确保津贴正常发放。

确认邮箱和手机

验证您的邮箱和手机号码，以便接收重要通知。

安全提示

  • 请勿将密码告知他人
  • 定期更换密码
  • 使用后及时退出系统
  • 不要在公共电脑上保存登录信息

→
查看任务详情
→
确认或拒绝任务
→
计入统计报表

图：任务处理流程图

操作步骤详解

  1. 查看任务

    登录系统后，在"我的任务"页面查看待处理的任务列表。任务会显示标题、发布时间、截止时间和状态。

  2. 查看详情

    点击任务标题或"查看详情"按钮，进入任务详情页面。仔细阅读任务描述、要求和相关附件。

  3. 确认任务

    如果接受任务，点击"确认"按钮。系统会记录您的接受时间，并开始计入工时统计。

  4. 拒绝任务

    如果因故无法接受任务，点击"拒绝"按钮并填写拒绝原因。拒绝的任务不会计入统计。

重要提醒

请在任务截止时间前完成确认或拒绝操作，逾期未处理的任务将自动标记为"已过期"。
